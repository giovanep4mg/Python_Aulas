Exercícios de Python 1

1. Variáveis e Tipos de Dados

Exercício 1: Crie um programa que declare três variáveis (nome, idade, altura) e imprima os valores.

Exercício 2: Converta um número decimal para inteiro e imprima o resultado.

Exercício 3: Peça ao usuário um número e mostre o dobro desse número.

2. Entrada e Saída de Dados

Exercício 4: Peça ao usuário seu nome e idade e imprima uma mensagem personalizada.

Exercício 5: Peça dois números ao usuário e exiba a soma deles.

Exercício 6: Peça um número ao usuário e exiba o quadrado desse número.

3. Operadores Matemáticos

Exercício 7: Peça três números ao usuário e calcule a média deles.

Exercício 8: Calcule a área de um círculo com base no raio fornecido pelo usuário.

Exercício 9: Converta um valor de reais para dólares com base na cotação informada.

Exercício 10: Peça um número e informe se ele é par ou ímpar.

4. Estruturas Condicionais (if/else)

Exercício 11: Peça um número e diga se ele é positivo, negativo ou zero.

Exercício 12: Verifique se um ano informado pelo usuário é bissexto.

Exercício 13: Peça dois números e informe qual é o maior.

Exercício 14: Peça a idade do usuário e diga se ele pode votar.

Exercício 15: Peça ao usuário uma senha e valide se é igual a "1234".

5. Estruturas de Repetição (for/while)

Exercício 16: Imprima os números de 1 a 10 usando um loop for.

Exercício 17: Peça um número e exiba sua tabuada de 1 a 10.

Exercício 18: Some os números de 1 a 100 usando um loop for.

Exercício 19: Continue pedindo números ao usuário até que ele digite um número negativo.

Exercício 20: Peça ao usuário um número e verifique se ele é primo.
