Exercícios de Manipulação de Dados e Funções em Python

Listas e Tuplas

1. Crie uma lista com cinco números e exiba o maior valor.
2. Crie uma tupla com três nomes e exiba o segundo nome.
3. Inverta a ordem dos elementos de uma lista.
4. Adicione um novo elemento a uma tupla sem modificar a original.
5. Concatene duas listas e exiba o resultado.

Dicionários e Conjuntos

1. Crie um dicionário com três pares chave-valor e exiba um dos valores.
2. Remova um item de um dicionário e exiba o dicionário atualizado.
3. Crie um conjunto com cinco números e adicione um novo número.
4. Verifique se um valor existe dentro de um conjunto.
5. Encontre a interseção entre dois conjuntos.

Funções e Retorno de Valores

1. Escreva uma função que receba dois números e retorne a soma.
2. Crie uma função que retorne o quadrado de um número.
3. Escreva uma função que receba um nome e retorne uma saudação personalizada.
4. Crie uma função que receba uma lista de números e retorne a média.
5. Faça uma função que verifique se um número é primo.

Manipulação de Strings

1. Converta uma string para maiúsculas.
2. Remova os espaços extras de uma string.
3. Substitua todas as ocorrências de uma palavra dentro de uma string.
4. Inverta os caracteres de uma string.
5. Verifique se uma string é um palíndromo.
