Estruturas de Repetição em Python

🎯 Objetivo da Aula

  • Compreender as estruturas de repetição for e while.
  • Aprender a utilizá-las para criar loops.
  • Testar no VS Code.

🔄 Estrutura de Repetição: for

O laço for é utilizado para iterar sobre sequências (listas, strings, range, etc.).

Exemplo 1: Iterando sobre uma lista

nomes = ["Ana", "Carlos", "João"]
for nome in nomes:
    print(f"Olá, {nome}!")

Exemplo 2: Usando range()

for i in range(5):
    print(i)

🔄 Estrutura de Repetição: while

O laço while executa um bloco de código enquanto uma condição for verdadeira.

Exemplo 1: Contador simples

contador = 0
while contador < 5:
    print(f"Contador: {contador}")
    contador += 1

Exemplo 2: Entrada do usuário

resposta = ""
while resposta != "sair":
    resposta = input("Digite algo (ou 'sair' para encerrar): ")
    print(f"Você digitou: {resposta}")

🚀 Rodando o Código no VS Code

  1. Abra o VS Code.
  2. Crie um arquivo .py (exemplo: repeticoes.py).
  3. Escreva o código acima.
  4. Salve o arquivo (Ctrl + S).
  5. Abra o terminal no VS Code (Ctrl + J).
  6. Execute o código digitando: python repeticoes.py

📌 Resumo

  • for é usado para iterar sobre sequências.
  • while executa enquanto a condição for verdadeira.
  • Para evitar loops infinitos, sempre garanta que a condição de parada seja atingida.
