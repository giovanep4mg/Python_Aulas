Aula sobre Funções e Retorno de Valores em Python

O que são Funções?

Funções são blocos de código reutilizáveis que executam uma tarefa específica. Elas ajudam a organizar o código e evitam repetições.

Definição de uma função

def saudacao():
    print("Olá, seja bem-vindo!")

saudacao()  # Chamada da função

Passando Parâmetros

Podemos passar valores para uma função através de parâmetros.

def cumprimentar(nome):
    print(f"Olá, {nome}!")

cumprimentar("Carlos")  # Saída: Olá, Carlos!

Retorno de Valores

Uma função pode retornar um valor usando a palavra-chave return.

def soma(a, b):
    return a + b

resultado = soma(5, 3)
print(resultado)  # Saída: 8

Parâmetros Padrão

Podemos definir valores padrão para os parâmetros de uma função.

def potencia(base, expoente=2):
    return base ** expoente

print(potencia(3))  # Saída: 9 (3^2)
print(potencia(2, 3))  # Saída: 8 (2^3)

Funções com Múltiplos Retornos

Uma função pode retornar múltiplos valores.

def calcular_dobro_triplo(numero):
    return numero * 2, numero * 3

dobro, triplo = calcular_dobro_triplo(4)
print(dobro)  # Saída: 8
print(triplo)  # Saída: 12

Conclusão

As funções são essenciais para organizar e reutilizar código. Elas permitem a criação de programas mais eficientes e fáceis de manter.
