Aula sobre Manipulação de Strings em Python

O que são Strings?

Strings são sequências de caracteres utilizadas para armazenar e manipular texto em Python. Elas podem ser definidas com aspas simples (' ') ou duplas (" ").

Exemplo de String

texto = "Olá, mundo!"
print(texto)  # Saída: Olá, mundo!

Principais Operações com Strings

Concatenação

Podemos unir duas ou mais strings utilizando o operador +.

nome = "Carlos"
sobrenome = "Silva"
nome_completo = nome + " " + sobrenome
print(nome_completo)  # Saída: Carlos Silva

Repetição

É possível repetir uma string utilizando o operador *.

risada = "ha" * 5
print(risada)  # Saída: hahahahaha

Acesso a Caracteres

Podemos acessar caracteres de uma string utilizando índices.

palavra = "Python"
print(palavra[0])  # Saída: P
print(palavra[-1])  # Saída: n (último caractere)

Fatiamento de Strings

Podemos extrair partes de uma string utilizando a sintaxe [início:fim:passo].

frase = "Manipulação de Strings"
print(frase[0:11])  # Saída: Manipulação
print(frase[12:])  # Saída: de Strings
print(frase[::-1])  # Saída: sgnirtS ed oãçalupinaM (string invertida)

Métodos Úteis para Strings

Transformação de Maiúsculas e Minúsculas

texto = "Python é incrível"
print(texto.upper())  # Saída: PYTHON É INCRÍVEL
print(texto.lower())  # Saída: python é incrível
print(texto.title())  # Saída: Python É Incrível

Remoção de Espaços

mensagem = "  Olá, mundo!  "
print(mensagem.strip())  # Saída: "Olá, mundo!"
print(mensagem.lstrip())  # Saída: "Olá, mundo!  "
print(mensagem.rstrip())  # Saída: "  Olá, mundo!"

Substituição de Caracteres

frase = "Eu gosto de Java."
nova_frase = frase.replace("Java", "Python")
print(nova_frase)  # Saída: Eu gosto de Python.

Quebra de Strings

Podemos dividir uma string em uma lista de palavras usando o método split().

texto = "Aprender Python é divertido"
palavras = texto.split()
print(palavras)  # Saída: ['Aprender', 'Python', 'é', 'divertido']

Junção de Strings

Podemos juntar elementos de uma lista em uma única string usando join().

palavras = ["Python", "é", "legal"]
frase = " ".join(palavras)
print(frase)  # Saída: Python é legal

Conclusão

A manipulação de strings é uma habilidade essencial em Python. Com essas técnicas, podemos trabalhar eficientemente com textos, processar entradas de usuários e modificar conteúdos dinamicamente.
