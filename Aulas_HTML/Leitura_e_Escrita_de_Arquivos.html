📂 Leitura e Escrita de Arquivos em Python

📌 1. O que é Manipulação de Arquivos?

Em Python, podemos ler e escrever arquivos usando a função open(). Isso permite armazenar e recuperar informações facilmente.

📝 2. Abrindo um Arquivo

arquivo = open("exemplo.txt", "r")  # Abre o arquivo para leitura
conteudo = arquivo.read()  # Lê o conteúdo do arquivo
print(conteudo)
arquivo.close()  # Fecha o arquivo

Modos de abertura:

  • "r": leitura
  • "w": escrita (sobrescreve o arquivo)
  • "a": escrita (adiciona ao final do arquivo)
  • "x": cria um novo arquivo

✏️ 3. Escrevendo em Arquivos

arquivo = open("exemplo.txt", "w")  # Abre para escrita
arquivo.write("Olá, mundo!\n")  # Escreve no arquivo
arquivo.write("Python facilita a manipulação de arquivos.\n")
arquivo.close()

📚 4. Lendo Linha por Linha

arquivo = open("exemplo.txt", "r")
for linha in arquivo:
    print(linha.strip())  # Remove quebras de linha extras
arquivo.close()

🚀 5. Usando "with open" (Forma Recomendada)

with open("exemplo.txt", "r") as arquivo:
    conteudo = arquivo.read()
    print(conteudo)  # O arquivo é fechado automaticamente

🔥 6. Conclusão

  • O método open() permite manipular arquivos.
  • Os modos "r", "w", "a", e "x" definem como o arquivo será manipulado.
  • Usar with open() é a forma mais segura de manipular arquivos.
